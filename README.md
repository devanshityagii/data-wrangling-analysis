# Data Wrangling and Trend Analysis
Analysis of 2013 hourly meteorological data from LaGuardia Airport (LGA), New York.

What this project covers:

Loading and exploring a real-world heterogeneous dataset
Data cleaning and unit standardisation (Fahrenheit to Celsius, mph to m/s, inches to mm)
Filtering, grouping, and aggregating data using Pandas
Identifying patterns and outliers across daily wind speed records
Time-series visualisation using Matplotlib
Communicating findings clearly through structured reporting

Tools: Python, Pandas, Matplotlib, Jupyter Notebook
